# Borderland Games 


## Game Theme Description
Borderland Games is a survival game inspired by the show Alice in Borderland, where players wake up in an abandoned Tokyo with only 7 visa days to their name. To survive, players must enter deadly arenas organised by card suit — Spades test physical strength, Diamonds test intelligence, Clubs test teamwork, and Hearts test psychological resolve. Each game won earns visa days while each loss costs them, and players can visit the shop to purchase survival items that give them an edge in the arenas. The goal is simple: win games, manage your visa days, and outlast everyone else in Borderland.

## Endpoints

###  Users
- `POST /users` — Register a new player into Borderland with 7 visa days (Currency in the game)
- `GET /users` — View all players currently in Borderland
- `GET /users?suit_affinity=diamonds` — Filter players by suit affinity (Suits are Diamonds,Spades,Clubs,Hearts)
- `GET /users/:id` — Get a specific player's profile and stats
- `PATCH /users/:id` — Update a player's username or suit affinity
- `DELETE /users/:id` — Eliminate a player from Borderland

###  Shop
- `GET /shop` — Browse all survival items available for purchase
- `GET /shop?suit=diamonds` — Filter shop items by suit
- `POST /shop/buy` — Buy an item using visa days, adds it to inventory

###  Inventory
- `GET /inventory/:user_id` — View all items a player is currently carrying
- `DELETE /inventory/:inventory_id` — Discard an item from inventory

###  Arenas (4 mini games)
- `GET /arenas` — View all 4 available game arenas
- `GET /arenas?suit=spades` — Filter arenas by suit
- `POST /arenas/arena_spades_3/play` — Play Osmosis, stamina vs random enemy strength
- `POST /arenas/arena_diamonds_5/play` — Play Blackjack, beat the dealer without busting
- `POST /arenas/arena_clubs_4/play` — Play Tug of War, your team level vs enemy team
- `POST /arenas/arena_hearts_7/play` — Play Solitary Confinement, cooperate or betray
- `GET /arenas/history/:user_id` — View a player's full game history and record

## Assumptions

- **Visa days as currency** — Visa days serve as both the player's life and in-game currency. Dropping to 0 prevents the player from entering any arena.
- **Game auto-resolution** — All 4 games are auto-resolved by the server. No real-time interaction is required beyond the initial API call.
- **Items are passive** — Items in inventory are automatically applied when entering an arena. Players do not need to manually equip them.
- **Blackjack simplified** — Cards are random numbers between 1-10 instead of a full 52-card deck. Aces are not treated as 1 or 11.
- **Tug of War enemy team** — The enemy team strength is randomly generated between 10-50 to simulate an opposing team without requiring multiple real players.
- **Solitary Confinement partner** — The partner's choice is randomly generated by the server. By default it is 50/50, but the Poker Face Mask shifts it to 65% cooperate.
- **No authentication for CA1** — User registration does not include login or session handling. I will add in CA2
- **Passwords stored as plaintext** — Passwords are stored without hashing for CA1. I will add Bcrypt hashing in CA2.
- **Fixed shop inventory** — The shop contains 5 fixed items seeded into the database. New items cannot be added
